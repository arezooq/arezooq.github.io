Intrusion Detection System

A Python-based system for detecting cyberattacks with integrated Machine Learning

Key Features

  • Real-time monitoring of network traffic
  • Custom datasets for training the ML model
  • Optimized algorithms for low hardware requirements
  • Detailed reports in multiple formats (e.g., PCAP, JSON)
  • Simulated cyberattacks for comprehensive testing

Technologies Used

Programming Languages:

Python

Frameworks:

Scikit-learn, NumPy, Pandas

Tools:

Wireshark, VMware

Database:

MongoDB

Project Objectives

  • Developing a lightweight and efficient intrusion detection system
  • Achieving minimal error rates in detecting various types of attacks
  • Providing scalable, accurate, and cost-effective network security solutions
