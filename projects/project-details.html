Intrusion Detection System

A Python-based system for detecting cyberattacks with ML integration

About the Project

The Intrusion Detection System is a cybersecurity tool designed to identify and analyze network intrusions in real-time. By integrating machine learning algorithms, this system provides accurate and efficient detection of various types of cyberattacks.

This project includes the following key features:

  • Real-time monitoring of network traffic
  • Custom datasets for training the ML model
  • Optimized algorithms for low hardware requirements
  • Detailed reports in multiple formats (e.g., PCAP and JSON)
  • Simulated cyberattacks for testing purposes

Technologies Used

  • Programming Languages: Python
  • Frameworks: Scikit-learn, NumPy, Pandas
  • Tools: Wireshark, VMware
  • Database: MongoDB

Project Objectives

The main goals of this project include:

  • Developing a lightweight intrusion detection system
  • Achieving minimal error rates in detecting attacks
  • Providing scalable and efficient solutions for network security
